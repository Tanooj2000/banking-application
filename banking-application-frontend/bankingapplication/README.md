# bankingapplication (Main Frontend App)

Main user/admin banking frontend built with React + Vite.

## Quick Start
1. Install dependencies:
	- `npm install`
2. Run development server:
	- `npm run dev`
3. Build production assets:
	- `npm run build`
4. Preview production build:
	- `npm run preview`
