# root-admin (Root Admin Frontend App)

Dedicated root-admin portal built with React + Vite.

## Quick Start
1. Install dependencies:
	- `npm install`
2. Run development server:
	- `npm run dev`
3. Build production assets:
	- `npm run build`
4. Lint code:
	- `npm run lint`
5. Preview production build:
	- `npm run preview`
