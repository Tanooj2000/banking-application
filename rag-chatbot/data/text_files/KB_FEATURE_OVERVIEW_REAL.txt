# InterBankHub - Feature Overview

Complete overview of all InterBankHub features for users and admins.

---

## Platform Overview

**What is InterBankHub?**

InterBankHub is a digital banking platform that enables customers to create and manage bank accounts across three countries:
- India
- USA  
- UK

Through a user-friendly interface, you can discover banks, complete account applications, and manage all your accounts in one place.

### 3. Account Creation

**Multi-Step Form Process:**
Complete 6-step wizard to create new accounts:

**Step 1: Select Branch**
- Choose your bank branch
- Location-based options
- Only available branches shown

**Step 2: Personal Details**
- Full name, date of birth, gender
- Contact information (email, phone)
- Address
- Country-specific ID numbers:
  - **India:** Aadhaar, PAN
  - **USA:** Social Security Number (SSN)
  - **UK:** National Insurance Number (NIN)

**Step 3: Educational Details**
- Education level (country-specific options)
- Institution name
- Course/degree
- Year of completion
- Grade/GPA (optional)

**Step 4: Income Details**
- Employment status (6 options)
- Employer name (if applicable)
- Occupation
- Monthly and annual income
- Income source (5 options)
- Currency varies by country (INR, USD, GBP)

**Step 5: Nominee Details**
- Nominee full name
- Relationship to applicant
- Nominee date of birth
- Contact information
- Address

**Step 6: Documents & Account**
- Upload 4 documents:
  - ID Proof
  - Address Proof
  - Income Proof
  - Passport Photo
- Select account type
- Enter initial deposit
- Accept terms & conditions

**Account Types Available:**

**India (INR):**
- Savings Account
- Current Account
- Salary Account
- Fixed Deposit

**USA (USD):**
- Checking Account
- Savings Account
- Money Market Account
- Certificate of Deposit

**UK (GBP):**
- Current Account
- Savings Account
- ISA (Individual Savings Account)
- Fixed Term Account

## Account Creation Flow

### Complete User Journey:

1. **Sign Up**
   - Create user account
   - Set password
   - Set country

2. **Log In**
   - Enter credentials
   - Access dashboard

3. **Browse Banks**
   - Select country
   - Search/filter banks
   - Browse available banks
   - Select bank and branch

4. **Create Account**
   - Click "Create Account"
   - Fill 6-step form
   - Upload documents
   - Submit application

5. **Wait for Approval**
   - Admin reviews application
   - Status shows as "Pending"
   - Typically 2-5 business days

6. **Receive Decision**
   - Check account status in dashboard
   - View approval or rejection reason
   - Status updates automatically

7. **Start Using Account**
   - Account approved
   - Download documents
   - Manage account

---

## Admin Application Processing Flow:

### Complete Admin Journey:

1. **Sign Up/Log In**
   - Create admin account or log in
   - Access Admin Dashboard

2. **View Pending Applications**
   - Dashboard shows pending count
   - Click to view pending applications
   - List displays all waiting applications

3. **Review Application**
   - Click on application
   - View all applicant details
   - Review all documents
   - Check for completeness and accuracy

4. **Make Decision**
   - Approve if all good
   - Reject if issues found
   - Optionally add reason for rejection

5. **Application Complete**
   - Application moves to Approved or Rejected section
   - Applicant can view status in their dashboard
   - Record maintained for future reference

## Integration Points

**What Connects to InterBankHub:**

1. **User Service** (localhost:8081)
   - User authentication
   - User profile management
   - User data storage

2. **Admin Service** (localhost:8083)
   - Admin authentication
   - Admin profile management
   - Admin data storage

3. **Bank Service** (localhost:8082)
   - Bank and branch information
   - Country and location data
   - Bank details and listings

4. **Account Service** (localhost:8085)

   - Document storage
   - Application management

## API Endpoints Used

### User Endpoints:
- Sign up user: POST /api/user/signup
- Sign in user: POST /api/user/signin
- Get user: GET /api/user/{userId}
- Update user: PUT /api/user/{userId}
- Change password: POST /api/user/changePassword

### Admin Endpoints:
- Sign up admin: POST /api/admin/signup
- Get admin: GET /api/admin/{adminId}
- Update admin: PUT /api/admin/{adminId}
- Change admin password: POST /api/admin/changePassword

### Bank Endpoints:
- Get countries: GET /api/bank/countries
- Get banks: GET /api/bank/{country}
- Get bank details: GET /api/bank/{bankId}
- Create branch: POST /api/bank/branch

### Account Endpoints:
- Create account: POST /api/account/create
- Get user accounts: GET /api/account/user/{userId}
- Get all accounts: GET /api/account/all
- Approve account: POST /api/account/{accountId}/approve
- Reject account: POST /api/account/{accountId}/reject
- View document: GET /api/account/document/{docId}
- Download document: GET /api/account/download/{docId}

### Chatbot Endpoint:
- Send message: POST /chat
