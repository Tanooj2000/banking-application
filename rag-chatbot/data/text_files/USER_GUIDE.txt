ACCOUNT CREATION GUIDE (User)

How to Create a New Bank Account
Route: /createaccount
1. Log in to your user account and go to the “Create Account” page (/createaccount).
2. The account creation process is a multi-step form. Complete each step as follows:
	 • Step 1: Select Branch (Route: /createaccount, first step)
		 – Choose your preferred bank branch from the available list.
	 • Step 2: Personal Details (Route: /createaccount, second step)
		 – Enter your full name, date of birth, gender, mobile number, email, address, Aadhaar number, and PAN number (all required).
	 • Step 3: Educational Details (Route: /createaccount, third step)
		 – Fill in your highest qualification and other educational information as prompted.
	 • Step 4: Income Details (Route: /createaccount, fourth step)
		 – Provide your occupation, annual income, and related details.
	 • Step 5: Nominee Details (Route: /createaccount, fifth step)
		 – Add nominee information (name, relationship, contact details).
	 • Step 6: Documents Upload (Route: /createaccount, final step)
		 – Upload all required documents (e.g., ID proof, address proof, income proof) as per the instructions.
3. Each step validates your input before allowing you to proceed to the next.
4. After completing all steps, review your information and submit the form.
5. You will see a confirmation message if your account creation request is successful. If there are errors, they will be displayed for correction.
6. Once submitted, you can view the status of your account application from your dashboard (Route: /userpage).

Tips:
• Prepare all required documents in advance for a smooth process.
• Double-check your details for accuracy before submitting.
• If you encounter validation errors, correct the highlighted fields and try again.

ADMIN USER GUIDE


1. Admin Login
Route: /signin (select Admin)
• Go to the Sign In page (/signin).
• Select "Admin" as your user type if prompted.
• Enter your admin username/email and password.
• On successful login, you are redirected to the admin dashboard (/adminpage).
• If credentials are invalid, an error message is displayed.

2. Admin Dashboard Overview
Route: /adminpage
• After login, you access the admin dashboard with navigation for profile, accounts, applications, branches, and more.
• The sidebar/menu allows switching between different admin features.

3. Manage Profile
Route: /adminpage (Profile section)
• In the dashboard, open the profile section.
• Click the edit button or open the profile edit modal.
• Update your admin details (username, email, bank, country, etc.) as needed.
• Save changes. The system validates for required fields and uniqueness.
• A confirmation message appears if the update is successful.

4. Change Admin Password
Route: /adminpage (Profile section)
• In your profile section, select the option to change your password.
• Enter your current password, new password, and confirm the new password.
• Save changes. The system will confirm if the password was updated successfully.

5. Manage User Accounts
Route: /adminpage (Accounts/Applications section)
• Navigate to the "Accounts" or "Applications" section in the dashboard.
• View all user account applications, filter by status (e.g., Pending, Approved, Rejected).
• Click on an application to view details.
• Approve or reject applications as needed. The system updates the status and notifies the user.

6. Manage Branches
Route: /adminpage (Branches section)
• Go to the "Branches" section in the dashboard.
• View, add, or edit branch details as required.
• Fill in all required fields for new or updated branches and save changes.

7. Log Out
Route: /adminpage or any admin page
• Click the “Logout” button in the navigation bar or admin menu.
• Your session is cleared and you are redirected to the login page (/signin).

Tips:
• Always use a strong, unique password for your admin account.
• Regularly review pending applications and branch information for accuracy.
• Log out after using the application, especially on shared or public devices.

For further help, contact support or refer to the FAQ section if available.

BANKINGAPPLICATION USER GUIDE (User Features Only)


1. Register as a New User
Route: /signup
• Go to the Sign Up page (/signup).
• Fill in all required fields: username, email, phone number, password, country, and bank (if applicable).
• The system checks for unique and valid data. Errors are shown for invalid or duplicate entries.
• Submit the form. On success, you are redirected to the login page (/signin).

2. Log In
Route: /signin
• Go to the Sign In page (/signin).
• Enter your username or email and password.
• On successful login, you are taken to your user dashboard (/userpage).
• If credentials are invalid, an error message is displayed.

3. Edit Profile
Route: /userpage (Profile section)
• On your dashboard (/userpage), locate your profile section.
• Click the edit button or open the profile edit modal.
• Update your username, email, or phone number as needed.
• Save your changes. The system validates for required fields and uniqueness.
• A confirmation message appears if the update is successful.

4. Change Password
Route: /userpage (Profile section)
• In your profile section, select the option to change your password.
• Enter your current password and your new password.
• Save changes. The system will confirm if the password was updated successfully.

5. View Linked Bank Accounts
Route: /userpage
• On your dashboard, view the list of your linked bank accounts.
• Click on an account to see details or associated documents.

6. Create a New Bank Account
Route: /createaccount
• Go to the “Create Account” page (/createaccount).
• Complete each step of the multi-step form: select branch, enter personal, educational, income, and nominee details, and upload documents.
• Each step validates your input before proceeding.
• Submit the form to create your account. Success or error messages are shown as appropriate.

7. Browse Banks and Branches
Route: /browsebank
• Go to the “Browse Bank” page (/browsebank).
• Search and filter banks by country, city, or branch.
• View details and images for each bank/branch.
• If logged in, see which banks you already have accounts with.

8. Log Out
Route: /userpage or any user page
• Click the “Logout” button in the navigation bar or user menu.
• Your session is cleared and you are redirected to the login page (/signin).

Tips:
• Always use a strong, unique password.
• If you encounter errors, check for validation messages and ensure all required fields are filled.
• Log out after using the application, especially on shared devices.

For further help, contact support or refer to the FAQ section if available.
